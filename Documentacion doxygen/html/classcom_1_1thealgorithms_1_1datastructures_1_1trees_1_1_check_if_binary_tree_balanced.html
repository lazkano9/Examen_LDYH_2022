Detailed Description

This class will check if a BinaryTree is balanced. A balanced binary tree is defined as a binary tree where the differenced in height between the left and right subtree of each node differs by at most one.

This can be done in both an iterative and recursive fashion. Below, isBalancedRecursive() is implemented in a recursive fashion, and isBalancedIterative() is implemented in an iterative fashion.
