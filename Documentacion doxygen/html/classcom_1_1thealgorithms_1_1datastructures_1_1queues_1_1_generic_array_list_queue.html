Detailed Description

This class implements a GenericArrayListQueue.

A GenericArrayListQueue data structure functions the same as any specific-typed queue. The GenericArrayListQueue holds elements of types to-be-specified at runtime. The elements that are added first are the first to be removed (FIFO). New elements are added to the back/rear of the queue.
